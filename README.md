# NIST 800-30 Risk Assessment & Threat Modeling

## Project Overview
This repository contains a semi-quantitative risk assessment and threat modeling simulation conducted in accordance with the NIST 800-30 to identify high-likelihood threat scenarios and prioritize remediation efforts for a simulated enterprise network, utilizing the likelihood times impact scoring method to allow leadership to prioritize identified risks, following the NIST risk assessment process to ensure a repeatable and logical outcome.
